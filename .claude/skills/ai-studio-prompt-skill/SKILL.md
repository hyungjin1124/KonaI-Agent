# AI Studio Build 프롬프트 생성기

요구사항을 분석하여 Google AI Studio Build에 최적화된 프롬프트를 생성한다.

## 핵심 원칙

**자유도 우선**: 화면 유형별 템플릿을 사용하지 않는다. AI Studio Build가 요구사항을 해석하여 최적의 UI를 창의적으로 설계하도록 한다.

**최소 제약**: 다음 항목만 일관되게 적용한다:
- 한글 UI (레이블, 버튼, 메시지)
- 비즈니스 앱 톤 (전문적, 깔끔)
- 반응형 고려

## 프롬프트 생성 프로세스

### 1단계: 요구사항 분석

사용자 요구사항에서 다음을 추출한다:

| 추출 항목 | 설명 | 예시 |
|-----------|------|------|
| 핵심 목적 | 화면이 해결하는 문제 | "일일 매출 현황 파악" |
| 주요 데이터 | 표시/입력할 정보 | 매출액, 거래처, 제품별 실적 |
| 사용자 액션 | 수행 가능한 작업 | 조회, 필터링, 다운로드 |
| 특별 요청 | 명시적 UI 요구사항 | "차트 포함", "모바일 대응" |

### 2단계: 프롬프트 구성

다음 구조로 프롬프트를 작성한다:

```
## 화면 목적
[핵심 목적을 1-2문장으로]

## 주요 기능
[사용자가 이 화면에서 할 수 있는 것들]

## 데이터 요소
[표시하거나 입력받을 데이터 항목들]

## 사용자 상호작용
[버튼, 필터, 정렬 등 인터랙션]

## 스타일 방향
- 한글 인터페이스
- 비즈니스 애플리케이션 스타일
- [추가 스타일 힌트가 있다면]
```

### 3단계: 프롬프트 품질 체크

생성된 프롬프트가 다음을 만족하는지 확인:

- [ ] 목적이 명확한가
- [ ] 데이터 요소가 구체적인가
- [ ] 불필요한 제약이 없는가 (AI Studio Build 자유도 확보)
- [ ] 한글 UI 명시되었는가

## 프롬프트 작성 가이드라인

### DO (권장)
- 목적과 기능 중심으로 기술
- 필요한 데이터 항목 나열
- 사용자 관점에서 작성
- 구체적인 예시 데이터 포함 (realistic mock data)

### DON'T (금지)
- 특정 레이아웃 강제 (예: "좌측에 메뉴, 우측에 콘텐츠")
- 특정 컴포넌트 지정 (예: "Material UI Table 사용")
- 픽셀 단위 크기 지정
- 색상 코드 직접 지정 (톤만 제안)

### 스타일 힌트 (선택적)

요구사항에 특별한 스타일 언급이 없으면 기본값 사용:

| 상황 | 스타일 힌트 |
|------|-------------|
| 경영진 대상 | "executive dashboard 느낌, 핵심 지표 강조" |
| 데이터 입력 | "깔끔한 폼, 입력 가이드 명확히" |
| 모니터링 | "실시간 느낌, 상태 표시 명확히" |
| 기본값 | "전문적인 비즈니스 앱" |

## 출력 형식

프롬프트 생성 후 다음 형식으로 제공:

```markdown
# AI Studio Build 프롬프트

[생성된 프롬프트 내용]

---

**사용법**: 위 내용을 Google AI Studio Build에 붙여넣기
**예상 결과**: [간단한 예상 화면 설명]
**조정 포인트**: [AI Studio Build 결과를 보고 조정할 수 있는 부분]
```

## 예시

### 입력 (요구사항)
```
일일 손익 현황을 볼 수 있는 대시보드가 필요해요.
오늘 매출, 원가, 손익을 한눈에 보고 싶고,
시간대별 매출 추이도 보고 싶어요.
제품별로 어떤 게 잘 팔리는지도요.
```

### 출력 (프롬프트)
```markdown
# AI Studio Build 프롬프트

## 화면 목적
당일 영업 실적과 손익 현황을 실시간으로 파악하는 경영 대시보드

## 주요 기능
- 오늘의 핵심 지표(매출, 원가, 손익) 즉시 확인
- 시간대별 매출 흐름 파악
- 제품별 판매 실적 비교

## 데이터 요소
핵심 지표:
- 당일 총 매출: 125,400,000원
- 당일 총 원가: 98,200,000원  
- 당일 손익: 27,200,000원 (손익률 21.7%)

시간대별 매출 (예시):
- 09시: 8,500,000원
- 10시: 12,300,000원
- 11시: 15,800,000원
- ...

제품별 실적 (예시):
- A제품: 45,000,000원 (36%)
- B제품: 38,000,000원 (30%)
- C제품: 25,400,000원 (20%)
- 기타: 17,000,000원 (14%)

## 사용자 상호작용
- 조회 기간 선택 (오늘/이번주/이번달)
- 데이터 새로고침
- 상세 내역으로 드릴다운

## 스타일 방향
- 한글 인터페이스
- Executive dashboard 느낌
- 핵심 숫자가 눈에 띄게
- 긍정/부정 실적을 색상으로 구분

---

**사용법**: 위 내용을 Google AI Studio Build에 붙여넣기
**예상 결과**: KPI 카드 + 라인 차트 + 파이/바 차트 조합의 대시보드
**조정 포인트**: 차트 종류, 색상 톤, 카드 배치
```

## 고급: 연속 화면 생성

여러 화면을 연속으로 생성할 때:

1. 각 화면별로 별도 프롬프트 생성
2. 화면 간 네비게이션 흐름 명시
3. 일관된 스타일 톤 유지를 위해 "이전 화면과 동일한 스타일" 문구 추가
