# Templates

이 디렉토리는 KonaI Agent 프로젝트의 템플릿을 관리하는 공간입니다.

## 📁 하위 디렉토리

### prompts/
AI Studio Build에서 사용할 프롬프트 템플릿을 저장합니다.
- 에이전트 시스템 프롬프트
- 작업별 프롬프트 템플릿
- 프롬프트 체이닝 시나리오

### docx/
문서 자동 생성을 위한 스크립트와 템플릿을 저장합니다.
- 명세서 생성 스크립트
- 보고서 템플릿
- 문서 변환 유틸리티

## 🎯 용도

템플릿을 사용하여 일관성 있는 문서와 프롬프트를 생성하고 관리합니다.

## 🔧 사용 방법

각 하위 디렉토리의 README.md를 참조하세요.

## 📝 템플릿 작성 원칙

1. **재사용성**: 다양한 상황에서 재사용 가능하도록 작성
2. **명확성**: 사용 방법과 목적을 명확히 문서화
3. **유지보수성**: 쉽게 수정하고 확장할 수 있도록 구조화
4. **일관성**: 프로젝트 전체에서 일관된 스타일 유지
