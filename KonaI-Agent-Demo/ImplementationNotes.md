
# Manus AI UI Clone 구현 노트

## 1. 컴포넌트 인벤토리 (Component Inventory)
- **Sidebar**: 로고, 내비게이션(새 작업, 검색, 라이브러리), 프로젝트 관리, 작업 히스토리 목록, 크레딧 홍보 카드.
- **Header**: 모델 선택 드롭다운, 알림, 크레딧 잔액(549), 사용자 프로필.
- **Main Hero**: "무엇을 도와드릴까요?" 환영 메시지와 플랜 정보.
- **Chat Interface**: 텍스트 입력 영역(Textarea), 액션 버튼(파일 추가, 마이크, 전송), 하단 도구 연결 바(Chrome, Gmail 등 통합 아이콘).
- **Quick Action Chips**: 슬라이드 제작, 웹사이트 구축 등 주요 작업 바로가기 버튼들.
- **Promo Card**: 하단 캐러셀 형태의 풀스택 앱 구축 프로모션 카드.

## 2. 레이아웃 구조 (Layout Structure)
- **Flexbox Layout**: 사이드바(고정 너비)와 메인 콘텐츠(가변 너비)로 구성된 2컬럼 레이아웃.
- **Sticky Header**: 메인 콘텐츠 상단에 고정된 헤더.
- **Centered Content**: 채팅창과 환영 메시지는 중앙 정렬(`max-w-5xl`).
- **Responsive Design**: 모바일 환경을 고려하여 사이드바 토글 및 그리드 조정 적용.

## 3. 구현 상세 및 가정한 사항
- **색상 팔레트**: 다크 모드를 위해 `#0A0A0A`(배경), `#121212`(사이드바), `#1A1A1A`(카드/입력창) 계열의 다크 그레이 톤을 사용했습니다.
- **아이콘**: Lucide React 라이브러리를 사용하여 레퍼런스 이미지와 가장 유사한 아이콘들을 배치했습니다.
- **인터랙션**: 입력창에 텍스트가 있을 때만 전송 버튼이 활성화되는 로직을 추가했습니다.
- **서체**: 가독성이 좋은 Pretendard 또는 시스템 기본 폰트를 적용했습니다.
- **이미지**: 프로모션 카드의 이미지는 `picsum.photos`의 플레이스홀더를 사용했습니다.

## 4. 접근성 (Accessibility)
- 의미론적 HTML 태그(`aside`, `main`, `header`, `button`, `textarea`)를 사용했습니다.
- 적절한 명암비와 아이콘 텍스트 레이블을 고려했습니다.
